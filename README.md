##순서있는 목록(번호)
순서있는 목록은 숫자와 점을 사용하여 나타낼수 있음
ex) 1. 첫번째
    2. 두번째
    3. 세번째

##순서없는 목록(글머리 기호:*,+,-)
글머리 기호를 앞에 붙여 순서없는 목록을 작성가능
ex) *빨강
    *노랑
    *초록
글머리 기호 혼합하여 사용 가능
